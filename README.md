# 寮費管理システム

出稼ぎ・住み込みスタッフ向けの寮費管理Webアプリ。  
キャストごとの宿泊記録・支払い・残高をブラウザで一元管理できます。

## 機能

| ページ | 内容 |
|---|---|
| キャスト管理 | キャストの登録・編集・支払いURL発行 |
| 部屋管理 | 建物・部屋の登録・編集 |
| 宿泊記録 | チェックイン・アウトの管理 |
| 月額管理 | 月額キャストの請求額を月ごとに管理 |
| 支払い確認 | 未確認支払いの承認・取り消し |
| 支払い入力 | キャスト専用ページ（認証不要・URL共有方式） |
| 残高確認 | 全キャストの請求額・支払い済み・残高サマリー |

### 料金体系

- **日額**: チェックイン〜アウト日数 × 1泊単価（キャストごとに設定）
- **月額**: 月ごとに金額を手動登録（出勤日数条件の管理あり）

### セキュリティ

- パスワード: PBKDF2-HMAC-SHA256（260,000イテレーション）+ ランダムsalt
- ログイン失敗3回で30分ロック
- セッション8時間で自動ログアウト
- 支払い入力ページのみ認証不要（キャスト専用URL）

## 技術スタック

| | |
|---|---|
| 言語 | Python 3.11+ |
| フレームワーク | Streamlit |
| DB | SQLite（`data/ryohi.db` — gitignore済み） |

## セットアップ

```bash
# リポジトリをクローン
git clone https://github.com/A-book1824215/ryohi-kanri.git
cd ryohi-kanri

# 依存パッケージをインストール
pip install -r requirements.txt

# 起動
streamlit run app.py
```

初回起動時に管理者パスワードの設定画面が表示されます。
